# Scramble Ghosts

## Build commands

- desktop: `go build --gcflags '-N -l' -o scramble_ghosts_X.X.X .`
- WASM: `env GOOS=js GOARCH=wasm go build --gcflags '-N -l' -o scramble_ghosts_X.X.X.wasm .`
